## Setup Instructions

1. **Clone & Install**:

   ```bash
   git clone <repo-url>
   cd contribute
   pnpm install
   ```

2. **Environment Setup**:
   Create a `.env` file with:

   ```env
   DATABASE_URL="your-postgresql-url"
   GITHUB_TOKEN="your-github-pat"
   GOOGLE_GENERATIVE_AI_API_KEY="your-gemini-key"
   ```

3. **Database Sync**:

   ```bash
   npx prisma db push
   npx prisma generate
   ```

4. **Run Development Server**:
   ```bash
   pnpm run dev
   ```
